Metodología CRISP-ML aplicada

1. Comprensión del negocio

Se busca anticipar riesgos sanitarios en municipios como Bogotá, Chía, Cajicá, Sopó y Zipaquirá mediante modelos predictivos de calidad del agua.

2. Comprensión de los datos

Se analizaron registros del IRCA con variables como pH, turbiedad, color, coliformes totales y fecales.

3. Preparación de los datos

Se limpiaron valores nulos y simulados datos adicionales para el entrenamiento del modelo.

4. Selección del modelo

Se eligió Gradient Boosting por su precisión y adaptabilidad a datos complejos.

5. Evaluación del modelo

Se utilizaron métricas como precisión, recall y matriz de confusión para validar el rendimiento.

6. Implementación

El modelo fue integrado a una aplicación Flask accesible vía web.

7. Mantenimiento y monitoreo

Se plantea mejorar el modelo con nuevos datos y monitorear su comportamiento.
